# Master’s Thesis: Monte Carlo Methods for Joint Simulation and Rendering

This repository contains the work developed for my **Master’s thesis in Computer Science** at  
**Friedrich-Alexander-Universität Erlangen-Nürnberg (FAU)**, completed at the **Chair of Computer Graphics**.

## Overview

The thesis focuses on **Monte Carlo algorithms** used in **physically based rendering**, particularly in the context of **path tracing** and **global illumination**. The goal of this work was to explore and implement alternative approaches that extend classical rendering techniques to support **volumetric simulation and rendering within a unified framework**.

The research investigates methods that enable efficient simulation and rendering of volumetric phenomena using stochastic techniques.

## Research Topics

The thesis explores several topics related to Monte Carlo rendering:

- **Path Tracing and Global Illumination**  
  Study and implementation of Monte Carlo algorithms used for physically based rendering of scenes.

- **Volumetric Path Tracing**  
  Investigation of alternative approaches capable of handling **light transport inside participating media**.

- **Shell Tracing for Homogeneous Volumes**  
  Analysis of the **Shell Tracing** technique and adaptation of the method for use with **homogeneous volumetric media**.

- **Joint Simulation and Rendering**  
  Exploration of how **physical simulation and rendering** can be integrated into a single framework and algorithm.

- **Walk-on-Spheres Simulation**  
  Research and implementation of the **Walk-on-Spheres** method within a Monte Carlo rendering framework.

- **Combining Walk-on-Spheres with Volumetric Path Tracing**  
  Investigation of approaches for integrating **Walk-on-Spheres simulation with volumetric rendering techniques**.

## Implementation

The practical implementation was developed in **C++** using a rendering framework created and maintained by the **Chair of Computer Graphics at FAU**.

The work included:

- Implementing **Monte Carlo rendering algorithms**
- Integrating **Walk-on-Spheres simulation**
- Extending the framework with **additional data structures and rendering components**

All implementations were built on top of the existing framework used for research and teaching at the Chair of Computer Graphics.
